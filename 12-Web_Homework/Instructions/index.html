Summary: Latitude vs. X

The project was designed to analyze different weather factors accross different latitudes around the world. To accomplish this we used the OpenWeatherMap API to pull data for over 500 cities.

We then used MatPlotLib to create scatter plots of the weather factors vs. latitude. Factors we looked at were Temperature, Humidity, Cloudiness and Windspeed. This site contains the visualizations, along with the source data and explinations.
